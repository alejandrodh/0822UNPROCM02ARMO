Micro desafío 2

  1. Creá un documento html, un archivo css y vinculalos.
  2. Dentro del html creá un título principal, 4 subtítulos, 2 párrafos para cada subtítulo y una lista desordenada con 6 elementos.
  3. Cambiá todas las tipografías que ve el usuario por Arial y que el color de fondo sea #eee
  4. Los subtítulos pares deben ser de color azul. Los subtítulos impares deben ser de color rojo.
  5. Todos los párrafos deben tener tamaño de letra 16px.
  6. Los elementos de la lista deben tener un fondo de color verde translucido al 50% y estar en negrita.
  7. Tarea para el viernes: terminar los puntos que falta y aplicar el mismo color del subtítulo a los párrafos.
